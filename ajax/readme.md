AJAX Folder

This folder contains all server-side scripts that handle AJAX requests.

Purpose:
- Used for asynchronous operations (no page reload required)
- Improves user experience by dynamically updating data

Examples:
- Fetching car data
- Submitting forms without refresh
- Updating availability status

Rules:
- Keep responses in JSON format where possible
- Do not include full HTML pages here
- Keep logic focused and lightweight
